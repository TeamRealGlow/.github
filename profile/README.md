## 6. 구현 핵심 기술
### AI 서버
- Instance Segmentation <br> Bisnet 를 통한 Segmentation 을 통해 얼굴을 인식하고 얼굴 부위 분할을 통해 세그멘테이션 맵 추출
<img src="img/faceSegment.png" alt="대체 텍스트" width="600">

- Image Processing<br>
이미지의 비트연산을 이용해 MASK를 추출 알파 블렌딩을 이용해 자연스럽게 색을 적용해 합성하는 알고리즘을 적용
<img src="img/Image Processing.png" alt="대체 텍스트" width="600">

### 프론트 웹
- Ajax 기반의 비동기 통신<br>Ajax를 활용한 비동기 호출을 통해 CSR(Client-Side Rendering) 방식의 데이터 렌더링을 구현. 서버 요청 시 페이지 리로드 없이 필요한 데이터만 갱신하여 사용자   경험(UX)을 향상시킴.

- React와 상태 관리 <br>
React의 컴포넌트 기반 구조를 사용해 UI를 효율적으로 관리, 데이터 흐름을 일관성 있게 유지함.

- 반응형 웹 디자인 <br>
Media querie를 이용해 모바일과 컴퓨터의 반응형 디자인 구현

- heic2any를 통한 이미지 호환<br>
모바일에서 많이사용하는 이미지 형식인 HEIC/HEIF의 처리를 위해 heic2any를 사용, 모바일 기기에서의 이미지 호환성을 높임

- browser-image-compression를 통한 이미지 압축<br>
고화질 이미지는 5MB가 넘는 일이 많으며, 서버의 부담을 키울 수 있어 browser-image-compression를 이용해 이미지를 압축해 서버에 부담을 최소화함.
